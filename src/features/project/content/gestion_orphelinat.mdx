## Overview

Gestion Orphelinat is a web application developed during my internship at **Souk El-Ahad Orphanage**.

The goal was to digitize internal processes and centralize information related to children, activities, and administrative reports.

## Technologies

- PHP
- MySQL
- MVC Architecture
- Bootstrap
- JavaScript
- HTML / CSS

## Key Features

### Children Management

- Add and edit children profiles
- Track personal information and history
- Organize records in a structured database

### Activities Management

- Register activities organized by the orphanage
- Associate children with activities
- Store historical activity data

### Reports

- Generate administrative reports
- Simplify internal monitoring

## Architecture

The project follows the **MVC architecture**:

- **Model** → Database logic (MySQL)
- **View** → Bootstrap interface
- **Controller** → Application logic

This structure helped maintain clean and organized code.

## Interface

The UI was built using **Bootstrap** to ensure:

- responsive layout
- simple navigation
- accessible management panels

## Deployment

The application was deployed internally and used by the orphanage staff for daily management tasks.

## Outcome

The system replaced manual processes and significantly simplified information management.
