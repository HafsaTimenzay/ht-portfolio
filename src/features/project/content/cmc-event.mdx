## Overview

CMC Event is a mobile application designed to centralize and promote events organized by **CMC**.

The app allows users to explore events while administrators can easily manage and publish new ones.

## Technologies

- Flutter
- Dart
- Supabase
- PostgreSQL
- Generative AI

## Features

### Event Feed

Users can browse a list of upcoming events with:

- title
- description
- date
- location

### AI Generated Content

The application integrates **Generative AI** to automatically generate:

- event descriptions
- promotional images

This reduces the workload for event organizers.

### Backend Infrastructure

The backend uses **Supabase** which provides:

- PostgreSQL database
- API layer
- authentication support

### Clean Mobile Interface

The application focuses on:

- modern mobile UI
- smooth navigation
- easy event discovery

## Architecture

The architecture consists of:

Frontend  
Flutter mobile application

Backend  
Supabase services + PostgreSQL database

AI Layer  
Automated generation of content
