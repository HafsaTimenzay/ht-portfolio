## Overview

JobMate is a platform designed to connect students with job opportunities such as:

- internships
- part-time jobs
- freelance work

## Features

- job listings
- filtering by field
- simple application system
- profile creation

## Technologies

- Web application architecture
- Frontend framework
- Backend API
- Database

## Goal

The platform aims to reduce the difficulty students face when searching for early career opportunities.
