## Overview

This project is a **full-stack website developed for a driving school**.

The goal was to create an online platform where students can access information about courses, services, and registration procedures.

## Technologies

Frontend

- React
- JavaScript
- CSS

Backend

- Express.js
- Node.js

## Features

### Course Information

The website presents:

- driving course categories
- training details
- pricing information

### Responsive Design

The interface is fully responsive and optimized for:

- desktop
- tablet
- mobile devices

### Structured Content

Information is organized into clear sections to make navigation simple for users.

### Contact & Information

Students can easily access:

- contact information
- location details
- registration instructions

## Development

The project followed a **full-stack approach**:

Frontend  
React-based interface

Backend  
Express.js API

## Result

The final product provides a professional online presence for the driving school and improves accessibility for potential students.
