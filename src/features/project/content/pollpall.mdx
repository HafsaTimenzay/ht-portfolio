## Overview

PollPal is a platform where users can create polls and gather opinions from others quickly.

The goal is to make decision-making easier through simple and interactive polls.

## Features

- Create polls
- Vote in polls
- View live results
- Share polls with others

## Technologies

- React
- Node.js
- REST API
- Database for storing polls and votes

## Use Cases

- community feedback
- quick decision making
- classroom interaction
